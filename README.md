# HABIBAH - E-commerce Tunisien (Zrir & Bsissa)

Projet e-commerce complet pour la marque **HABIBAH**, spécialisée dans les délices traditionnels tunisiens.

## 🚀 Liens du projet (Live)

- **Frontend (Site web)** : [https://stately-cuchufli-eb7aa3.netlify.app/](https://stately-cuchufli-eb7aa3.netlify.app/)
- **Backend (API)** : [https://habibah-api.onrender.com/](https://habibah-api.onrender.com/)
- **Base de données** : PostgreSQL sur [Neon.tech](https://neon.tech/)

## 🛠️ Stack Technique

- **Frontend** : React, Tailwind CSS, Framer Motion, i18next (FR/AR).
- **Backend** : Node.js, Express, PostgreSQL.
- **Authentification** : JWT, Google OAuth.
- **Emailing** : Nodemailer.

## 📦 Structure du projet

- `/frontend` : Application React.
- `/backend` : API Node.js et logique métier.

## 📂 Déploiement

- **Frontend** : Hébergé sur Netlify (connecté au dossier `/frontend`).
- **Backend** : Hébergé sur Render (connecté au dossier `/backend`).
- **Base de données** : Neon PostgreSQL.
